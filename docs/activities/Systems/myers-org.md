# Computer Organization (Myers)

Brandon Myers developed these activities for Architecture and Organization with support from an ACM Special Projects Award.
The collection is published on GitHub: [Guided Inquiry for Computer Organization](https://bmyerz.github.io/pogil-for-computer-organization/).

## Contents

1. Bits and numbers
2. Bytes and memory
3. Memory Organization of Programs
4. Stored Programs
5. Procedure Calls
6. Combinational logic
7. Adders and delay
8. Adders, shifters, multipliers
9. Sequential logic
10. Addressable memory
11. The Add Instruction
12. Engineering digital systems
