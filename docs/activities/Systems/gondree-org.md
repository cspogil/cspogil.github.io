# Computer Organization (Gondree)

These activities were developed by Mark Gondree for Architecture and Organization.
They focus on the 32-bit ARM architecture and are based on activities by Clif Kussmaul.
For full access, see this [Google Drive folder](https://drive.google.com/drive/u/0/folders/1toE65XTF5LlrDwJKiKpJdTogpRzDjBVH).

## Contents

1. Data Reps I - Bits
2. Data Reps II - Integers
3. Combinational Logic - Adders
4. Code Reps I - Architecture
5. Code Reps II - Steps & Files ARM
6. Code Reps III - Addressing ARM
7. Code Reps V - Control Flow ARM

The following activity was written during the {{AWP}}:
