# Operating Systems (Norman)

These activities were developed by Victor Norman based on the textbook Operating Systems Concepts Essentials by Silbershatz, Galvin, and Gagne.
The full versions are in this [Google Drive folder](https://drive.google.com/drive/folders/1tS1Hg0djC6Apzs3Lcy2ahl9hNMHOmJTQ).

## Contents

1. Intro to OS
2. System Calls
3. Processes
4. Producer-Consumer Problem
5. Critical Sections
6. Data Races
7. Single CPU Process Scheduling
8. Address Checking & Translation
9. Virtual Memory
10. Disk Scheduling Algorithms
