# CS2 in Java (Lemons)

These activities were developed by Sofia Lemons. Most activities include a lesson plan with introductory discussion prompts and end-of-class assessment questions implemented as [Kahoot](https://kahoot.com/) quizzes.
For sample versions or full access, contact Sofia Lemons.

## Contents

1. JAV_INT - Interfaces & Abstract Classes
2. JAV_INH - Inheritance
3. ALG_REC_2 - Control flow, tail recursion, & backtracking
4. ALG_SEARCH - Sequential & binary search
5. ALG_POST - Infix & postfix notation
6. ALG_TREE_1 - Trees
7. ALG_TREE_2 - Binary Search Trees
8. ALG_TREE_3 - Expression Trees & Traversals
9. ALG_TREE_4 - State Trees
10. ALG_TREE_5 - Game Trees
11. ALG_SORT_2 - bubble, selection, & insertion sort
