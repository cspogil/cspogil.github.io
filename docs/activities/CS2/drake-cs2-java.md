# Learn Java in N Games (Drake)

Peter Drake developed these activities for CS2 as part of the project _Learn Java in N Games_, which includes other (non-POGIL) activities.

* Activities are on GitHub at: [PeterDrake/ljing2-skeletons](https://github.com/PeterDrake/ljing2-skeletons)
* Solutions (access on request for faculty) are at: [PeterDrake/ljing2-solutions](https://github.com/PeterDrake/ljing2-solutions)

## Contents

| Activity                                           | Notes                                   |
| -------------------------------------------------- | --------------------------------------- |
| 01. Shut The Box - arrays, loops, teamwork         | EngageCSEdu Engagement Excellence Award |
| 04. Tic-Tac-Toe - Javadoc, 2D arrays, teamwork     | EngageCSEdu                             |
| 07. Anagrams - trace calls, JUnit, info processing | EngageCSEdu                             |
| 10. Towers of Hanoi - recursion, critical thinking | EngageCSEdu Engagement Excellence Award |
| 13. Beetle - classes, problem solving              | EngageCSEdu Engagement Excellence Award |

| 19. High Scores I - insertion sort, merge sort     |                                         |
| 22. Railyard - stacks, queues, interfaces          |                                         |
| 25. High Scores II - binary search, collections    |                                         |
| 28. Hex - graphs, depth-first, critical thinking   |                                         |
