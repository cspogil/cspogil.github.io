# CS1 in Java (Lemons)

These activities were developed by Sofia Lemons. Most activities include a lesson plan with introductory discussion prompts and end-of-class assessment questions implemented as [Kahoot](https://kahoot.com/) quizzes.
For sample versions or full access, contact Sofia Lemons.

## Contents

1. JAV_CODE - Code reading in Java
2. JAV_BRANCH_1 - Branching I
3. JAV_BRANCH_2 - If-Else Statements
4. JAV_ARR_2 - Arrays & Initialization
5. INTELLIJ_DEBUG_1 - Debugging Java in IntelliJ
6. INTELLIJ_DEBUG_2 - Debugging Java Methods in IntelliJ
7. ALG_THINKING - Algorithmic Thinking
8. DECOMP - Decomposition
9. PROB_SOLVING - Problem Solving Strategies
10. JAV_DES - Iterative Design
11. JAV_BIN - Binary Numbers
12. JAV_METH_1 - Methods Introduction
13. JAV_METH_2 - Methods & Parameters
14. JAV_MOD - Modularity
15. JAV_MIS - Common Misconceptions
