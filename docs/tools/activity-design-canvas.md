# Activity Design Canvas

The **Activity Design Canvas** (pdf) seeks to help authors via a one page overview of objectives, models, questions, & related info. It recognizes that design is inherently non-linear and moves among phases & perspectives. It is inspired by the [Business Model Canvas](https://www.strategyzer.com/library/the-business-model-canvas).

If you find this approach useful, please contact Clif Kussmaul (clif@kussmaul.org). I would love to see copies of canvases you have created, and any feedback you have, so we can improve the canvas design. Thank you!
